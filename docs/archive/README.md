# Arquivo Morto (Archive)

Este diretório contém documentos, análises e logs de refatoração que serviram seu propósito em momentos específicos do projeto mas que não refletem mais o estado atual ou não são mais necessários para o dia-a-dia.

Estão mantidos aqui para preservação histórica e consulta se necessário.

## Conteúdo Típico

- Logs de refatoração (`REFATORACAO_*.md`)
- Propostas antigas (`PROPOSTA_*.md`)
- Análises de incidentes ou inconsistências passadas (`ANALISE_*.md`)
