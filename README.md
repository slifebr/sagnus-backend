# Sagnus ERP — Plataforma Modular (DDD + Arquitetura Modular)

## Significado do nome SAGNUS

**SAGNUS** é um acrônimo de:

**S**istema de  
**A**rquitetura e  
**G**estão de  
**N**úcleos  
**U**nificados de  
**S**oftware  

O nome reflete exatamente o propósito da plataforma:

- múltiplos núcleos de negócio (Bounded Contexts)
- arquitetura modular orientada a domínio
- infraestrutura compartilhada padronizada
- evolução independente e escalável

SAGNUS simboliza um ERP moderno, organizado e coeso, onde cada módulo é um núcleo isolado e autônomo, mas que trabalha em conjunto por meio de contratos bem definidos.

## Visão geral

Esta versão do repositório contém apenas a estrutura base dos módulos:

- shared-kernel
- sagnus-shared-api-error
- sagnus-platform-security
- sagnus-bc-cadastro
- sagnus-bc-faturamento-nfe

A implementação de código será adicionada gradualmente seguindo as diretrizes de arquitetura em /docs.
